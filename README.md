# Figma Positivus - Flexbox Clone 🎨

A practice project for **Saylani** that recreates the famous Figma's Positivus design using modern HTML5, CSS3, and Flexbox layout techniques.

## 📋 Project Overview

This project is a learning exercise that demonstrates proficiency in front-end web development fundamentals. It's a pixel-perfect clone of the popular Positivus design template, built entirely from scratch using flexbox for responsive layout management.

**Key Focus Areas:**
- Flexbox layout techniques
- Responsive web design
- CSS3 styling and animations
- HTML5 semantic markup
- Design-to-code implementation

## 🎯 Objectives

- Master flexbox properties and layout patterns
- Practice translating Figma designs to responsive web pages
- Develop professional-grade HTML and CSS
- Create a fully responsive website that works across all devices
- Implement best practices in web development

## 🛠️ Technologies Used

- **HTML5** - Semantic markup structure
- **CSS3** - Styling and layout with Flexbox
- **Flexbox** - Primary layout system for responsiveness
- **JavaScript** - (Optional) for interactive elements if applicable

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)
- Code editor (VS Code, Sublime Text, etc.)
- Git installed on your system

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ansarwaseem/figma-positivus-by-ansar-waseem.git
```

2. Navigate to the project directory:
```bash
cd figma-positivus-by-ansar-waseem
```

3. Open `index.html` in your browser:
```bash
# For macOS
open index.html

# For Windows
start index.html

# Or simply double-click the index.html file
```

Alternatively, use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server package)
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
